# 🏏 IPL Data Analysis Dashboard

## 📌 Overview

This project focuses on analyzing IPL (Indian Premier League) data to uncover meaningful insights about team performance, player statistics, and match trends.

## 📊 Features

* Top Winning Teams Analysis
* Top Run Scorers (Batsmen)
* Top Wicket-Taking Bowlers
* Matches Played per Season
* Venue-wise Match Analysis
* Interactive Filters (Season, Team, Venue)

## 🛠 Tools & Technologies

* Power BI (Dashboard & Visualization)
* Python (Pandas, Matplotlib, Seaborn)
* Data Cleaning & Transformation

## 📈 Key Insights

* Mumbai Indians and Chennai Super Kings dominate IPL wins
* Virat Kohli is among the highest run scorers
* Yuzvendra Chahal leads in wicket-taking

## 🚀 Project Outcome

This project demonstrates:

* Data Cleaning & Preparation
* Data Visualization & Dashboard Design
* Business Insight Generation

## 📂 Files Included

* Clean datasets
* Jupyter notebooks
* Power BI dashboard (.pbix)
